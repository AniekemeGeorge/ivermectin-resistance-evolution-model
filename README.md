# Ivermectin Resistance Evolution Model

## Description
This repository contains MATLAB code for modelling resistance evolution under ivermectin-based interventions for different scenrios as listed in the corresponding files below.

## Files
- High_Resistance_Spread.m – simulations for high resistance scenarios  
- Medium_Resistance_Spread.m – medium spread scenarios  
- Low_Resistance_Spread.m – low spread scenarios  
- HumanSEI_VectorSI.m – transmission model Function  
- InvestigateR0.m – reproduction number analysis
-  SEIS_Human SEI_VectorSI - script to extablish the stability of the epidemiological system

## How to run
Run the main scripts in MATLAB (tested on version R2025b).
