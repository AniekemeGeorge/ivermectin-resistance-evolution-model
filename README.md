# Ivermectin Resistance Evolution Model

## Description
This repository contains MATLAB code for modelling resistance evolution under ivermectin-based interventions for different scenrios as listed in the corresponding files below.The model explores different resistance spread scenarios (low, medium, and high) and investigates how intervention dynamics influence resistance emergence and impact on malaria transmission.



## Files
- High_Resistance_Spread.m – simulations for high resistance scenarios  
- Medium_Resistance_Spread.m – medium spread scenarios  
- Low_Resistance_Spread.m – low spread scenarios  
- HumanSEI_VectorSI.m – transmission model Function  
- InvestigateR0.m – reproduction number analysis
-  SEIS_Human SEI_VectorSI - script to extablish the stability of the epidemiological system

## How to run
Run the main scripts in MATLAB (tested on version R2025b)
1. Open MATLAB
2. Navigate to the repository folder
3. Run one of the scenario scripts:
   - High_Resistance_Spread.m
   - Medium_Resistance_Spread.m
   - Low_Resistance_Spread.m

These scripts call the underlying transmission model functions.
